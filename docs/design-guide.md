# Design Guide — Haifa University Video Series

מדריך עיצוב ונראות לסדרת הסרטונים. כל סרטון חדש חייב לעקוב אחרי השפה הוויזואלית המתוארת כאן.

## שפה ויזואלית משותפת (כל הסרטונים)

### לוח צבעים

| Token | ערך | שימוש |
|-------|------|-------|
| bgPrimary | `#0a0e1a` | רקע כהה חיצוני (קצוות הגרדיאנט) |
| bgSecondary | `#111827` | רקע בהיר יותר (מרכז הגרדיאנט) |
| primary | `#00d4ff` | טורקיז — צבע ראשי, כותרות, גלואים |
| secondary | `#8b5cf6` | סגול — אלמנטים דקורטיביים, ניגודיות |
| accent | `#fbbf24` | זהב — הדגשות, מספרים חשובים |
| warning | `#ef4444` | אדום — שגיאות, אזהרות, הזיות |
| text | `#ffffff` | טקסט ראשי |
| textMuted | `#94a3b8` | טקסט משני |

| green | `#22c55e` | תקינות, מקורות נכונים |
| blue | `#3b82f6` | טקסט תומך, קטגוריות |

### טיפוגרפיה

- **פונט**: Rubik (דרך `@remotion/google-fonts`) — משקלים 300, 400, 600, 700, 800, subsets: hebrew + latin
- **כותרות ראשיות**: 64–100px, weight 800, עם text-shadow זוהר בצבע מתאים
- **כותרות משנה / סקשנים**: 42–54px, weight 600–700, בצבע primary
- **גוף טקסט**: 28–36px, weight 500–600
- **טקסט קטן / תגיות**: 18–24px, textMuted או opacity 0.75–0.85
- **letter-spacing**: 2–12px על טקסטים מודגשים (למשל "LLM" עם 12px)

### אפקט זוהר על טקסט (Text Glow)

כל כותרת חשובה מקבלת text-shadow כפול — גלו רך + גלו רחב:
```css
textShadow: "0 0 20px rgba(0, 212, 255, 0.5), 0 0 60px rgba(0, 212, 255, 0.3)"
```
הצבע משתנה לפי ההקשר (טורקיז למושגים טכניים, זהב להדגשות, אדום לאזהרות). לעתים הגלו פועם עם `Math.sin()`.

### RTL

- כל טקסט עברי: `direction: "rtl"` + `textAlign: "right"` על block-level container (`div`), **לא** על `span`
- טקסט אנגלי (LLM, AI, ChatGPT): `direction: "ltr"`
- Flexbox עם כרטיסים: `flexDirection: "row-reverse"` כדי שהקריאה תהיה מימין לשמאל

### רקע

כל שוט (למעט narrator video) כולל שכבות רקע:

1. **גרדיאנט רדיאלי**: `radial-gradient(ellipse at 50% 50%, ${bgSecondary} 0%, ${bgPrimary} 70%)`
2. **תמונת רקע AI** (אופציונלי): `<Img>` ב-opacity 0.25–0.35, עם fade-in ב-30–60 פריימים
3. **כהייה (vignette)**: `radial-gradient(ellipse, transparent 35%, rgba(0,0,0,0.25–0.35) 100%)`
4. **רשת נקודות** (Video 1): `radial-gradient(circle, ${textDim}22 1px, transparent 1px)` בגודל 40×40px, opacity 0.3

### גלסמורפיזם (Glassmorphism)

זהו הסגנון הדומיננטי לכרטיסים, פאנלים ומכולות תוכן:

```javascript
{
  background: "rgba(255, 255, 255, 0.05–0.08)",
  backdropFilter: "blur(16px)",
  border: `1.5px solid ${borderColor}44`,  // 27%–40% alpha
  borderRadius: 16–20,
  boxShadow: `0 0 30px ${borderColor}12, inset 0 1px 0 rgba(255,255,255,0.04)`,
}
```

- צבע הגבול משתנה לפי הקשר (טורקיז למושגים, אדום לשגיאות, זהב להדגשות, ירוק לתקינות)
- בהדגשה/hover: alpha עולה ל-0.55, shadow מתרחב

### אנימציות

**עיקרון מרכזי**: כל האנימציות דרך `useCurrentFrame()` + `spring()` / `interpolate()`. **אסור** להשתמש ב-CSS transitions או keyframes.

**Spring קונפיגורציה סטנדרטית** (כניסת אלמנטים):
```javascript
spring({
  frame: frame - startFrame,
  fps,
  config: { damping: 16, stiffness: 90, mass: 0.8 }
})
```
- damping 14–18: תנועה חלקה, לא קפיצית — **ללא bounce מוגזם**
- stiffness 80–100: מהירות מתונה (~300ms)
- mass 0.8: תגובה מהירה

**דפוסי כניסה**:
- **Scale + Opacity**: האלמנט גדל מ-0 ל-1 + opacity 0→1 (הכי נפוץ)
- **Translate + Opacity**: החלקה מחוץ למסך + fade in
- **Staggered delays**: אלמנטים ברצף נכנסים עם 20–60 פריימים הפרש

**אנימציות רציפות**:
- **פעימת גלו**: `Math.sin(frame * 0.05–0.08)` — נשימה של אור
- **סיבוב**: `frame * 0.15–0.8` — סיבוב רציף (גלובוס, סימן שאלה)
- **קורסור מהבהב**: `Math.sin(frame * 0.15–0.2) > 0` — מצב בינארי
- **נקודות הקלדה**: 3 נקודות עם `Math.sin(frame * 0.3 + offset)` staggered

**מה לא לעשות**:
- אין מעברים (transitions) בין שוטים — כל שוט מתחיל ומסתיים במקום
- אין אנימציות bounce מוגזמות — התנועה צריכה להיות מקצועית ורגועה
- אין slide-in/out או zoom בין שוטים

## Video 1 — "מה זה מודל שפה (LLM)?"

### אופי ויזואלי
סרטון עשיר באלמנטים ויזואליים מגוונים: SVG אנימטיביים, גלובוס תלת-ממדי, דיאגרמות flow, ממשק צ'אט, טרמינל. האסתטיקה היא "tech-forward" — מזכיר data visualization או particle simulation.

### אלמנטים ייחודיים

**חלקיקים צפים (Background.tsx)**:
- 30 חלקיקים מפוזרים בזווית הזהב
- צבעים מתחלפים: טורקיז, סגול, כחול כהה (`#1e40af`)
- גודל 2–6px עם glow shadow
- תנועה אנכית איטית + פעימת opacity מחזורית (300 פריימים)
- **רק ב-Video 1** — הסרטונים האחרים משתמשים בתמונות AI כרקע

**ממשק צ'אט (Shot 1.3)**:
- הודעת משתמש: מיושרת ימינה, רקע כחול כהה (`#1e3a5f`), פינות מעוגלות
- תשובת AI: מיושרת שמאלה, רקע טורקיז שקוף (`${primary}18`), typewriter
- נקודות typing: 3 נקודות עם opacity staggered

**טרמינל (Shot 3.1)**:
- רקע כהה (`#0d1117`), כרום (`#1a1a2e`)
- כפתורי traffic light (אדום/צהוב/ירוק)
- טקסט monospace עם קורסור מהבהב

**גלובוס wireframe (Shot 4.2)**:
- SVG עם meridians + parallels מסתובבים
- טקסט שפות צף סביב הגלובוס עם מיון עומק
- סיבוב: `frame * 0.8`

**חיצי flow (Shot 2.1)**:
- חיצים מקווקוים בטורקיז בין כרטיסים
- `strokeDasharray="8 4"` + שכבת glow מאחור
- ראש החץ מופיע ב-70% מהתקדמות

**פיזיקה (Shot 7.5)**:
- נפילת תפוח: `Easing.in(Easing.poly(2))` — האצה ריבועית (גרביטציה)
- נורה זוהרת: halo רדיאלי + פעימת brightness עם `Math.sin()`

### מוסיקת רקע
Video 1 הוא היחיד עם מוסיקת רקע (Suno AI, volume 0.10). סרטונים 2A ו-2B משתמשים רק בקריינות.

### לוגו
לוגו מוטמע כ-`<Img>` persistent ב-FullVideo.tsx (לא בתוך כל שוט). נעלם בהדרגה 30 פריימים לפני השוט האחרון.

## Video 2A — "כשהמודל נשמע משכנע אבל טועה" (Hallucinations)

### אופי ויזואלי
סרטון מובנה סביב **3 דוגמאות מעשיות** של הזיות AI. כל דוגמה מוצגת כממשק צ'אט → דקונסטרוקציה → אזהרה. אסתטיקה של "חקירה" — חשיפת טעויות בתוך מידע שנראה אמין.

### אלמנטים ייחודיים

**שינוי גוון רקע (Shot 1.1)**:
- תמונת brain-glitch עם `filter: hue-rotate()` דינמי
- מתחיל כחול (0deg), עובר לאדום (160deg) כשמגלים "Hallucinations"
- גלו אדום ambient פועם מאחור

**אפקט גליץ' על טקסט**:
- `Math.sin(frame * 2.5) * 2` — רעידה אופקית
- צבע מהבהב: `Math.sin(frame * 0.15) > 0 ? primary : warning`
- משמש במילה "Hallucinations" לאפקט דרמטי

**ממשק צ'אט + Typewriter**:
- כמו Video 1 אבל עם הדגשת קטעים (highlight):
  - ירוק (`rgba(34,197,94,0.3)`) למידע נכון
  - אדום (`rgba(239,68,68,0.3)`) למידע שגוי
- SVG checkmark עם stroke-dasharray animation

**פאנלים להשוואה (Shot 2.1)**:
- שני פאנלים זכוכיתיים זה לצד זה
- מסמך מדומה עם שורות צבעוניות ו-scan line שחושף סימני שגיאה (✗)
- `top: ${scanProgress}%` — קו סריקה נע מלמעלה למטה

**כרטיס אזהרה (Shot 5.2)**:
- רקע אדום שקוף (`rgba(239,68,68,0.08)`)
- משולש אזהרה SVG עם stroke draw animation
- shadow פועם: `20 + Math.sin(frame * 0.08) * 10`
- גלו אדום ambient ברקע השוט כולו

**סיום + טיזר (Shot 6.1)**:
- סימן שאלה SVG מסתובב ברקע (opacity 0.12, blur גאוסי)
- שלב סיום: לוגו אוניברסיטה על רקע לבן מעוגל

### קידוד צבע לפי מידע
- **ירוק** (#22c55e): מידע נכון, מקורות תקינים
- **אדום** (#ef4444): שגיאות, הזיות, מידע חסר
- **זהב** (#fbbf24): אזהרות קריטיות, תובנות מרכזיות

### לוגו
כמו Video 1 — `<Img>` persistent ב-FullVideo.tsx, נעלם לפני השוט האחרון. משתמש בלוגו ייעודי (`video2a/images/haifa-logo-white.png`).

## Video 2B — "למה מודלי שפה טועים לפעמים?"

### אופי ויזואלי
סרטון **הסברתי-אנליטי** — מסביר מנגנונים טכניים (איך LLM מייצר תשובה, למה הוא נשמע בטוח). עשיר ב-data visualization: גרפי הסתברות, מדד ביטחון, כרטיסי כללים. כל שוט מבוסס על **תמונת רקע AI** עם שכבות אנימציה מעל.

### אלמנטים ייחודיים

**סימן שאלה צף (Shot 1.1)**:
- סימן שאלה SVG ענק (500px) בסגול, מסתובב ברקע
- opacity 0.08, סיבוב: `frame * 0.15`
- יוצר אווירה של "חקירה" מההתחלה

**ויזואליזציית הסתברויות (Shot 2.1)**:
- גרפי bars שגדלים עם spring animation
- Bar מנצח מקבל גלו + gradient fill
- מספרי אחוזים גדולים וזוהרים
- "?" על bars לא ודאיים (בזהב)
- Badge הצלחה עם gradient רקע

**מד ביטחון (Shot 3.1)**:
- SVG חצי-עיגול עם gradient מירוק לזהב לאדום
- מחט שמתקדמת ל-100% (spring עם damping:30 — ציור איטי)
- סימן X אדום + "אין מנגנון ספק"
- רקע: תמונת מסכות תיאטרון (appearance vs. reality)

**Grid כללים 2×2 (Shot 4.1)**:
- 4 כרטיסים עם אייקוני SVG ייעודיים (זכוכית מגדלת, צ'קליסט, נורה, מגן)
- מספרים בעיגולי זהב
- תגיות צבעוניות (chips) מתחת לכרטיס 2
- הדגשת כרטיסים (brightness boost) בסוף השוט

**חלקיקים צפים (Shot 5.1)**:
- 8 חלקיקים בגדלים 4–8px (טורקיז, סגול, זהב)
- תנועה ורטיקלית עם `Math.sin(frame * speed * 0.01)`
- opacity נמוך (0.15–0.25) — אווירה מדיטטיבית
- **ללא כרטיסים או UI** — רק טקסט וחלקיקים למחשבה

**טיזר לסרטון הבא (Shot 5.2)**:
- כותרת "הטיות (Biases)" ב-80px סגול עם גלו פועם
- קווים דקורטיביים שגדלים מהמרכז
- חץ מונפש שמתנדנד (`Math.sin(frame * 0.05) * 5px`)
- vignette קולנועית
- סיום: לוגו על רקע לבן עם פינות מעוגלות

**Typewriter + Word Bubbles (Shot 2.1)**:
- שאלה מוקלדת תו-תו עם קורסור מהבהב
- בועות מילים שמודגשות בזמן האנימציה (border 1px + padding)

### קידוד צבע לפי קטגוריה
- **טורקיז** (#00d4ff): מושגים ראשיים, LLM, אימות
- **סגול** (#8b5cf6): נתיבים חלופיים, ביטחון, טיזרים
- **זהב** (#fbbf24): שאלות, אחריות, הדגשות
- **אדום** (#ef4444): שלילה, חוסר ודאות, מידע מסוכן
- **ירוק** (#22c55e): נכונות, אישור

### לוגו
משתמש ב-`<Logo />` component (מ-`design/Logo.tsx`) שמרונדר ב-FullVideo.tsx מעל כל השוטים.
